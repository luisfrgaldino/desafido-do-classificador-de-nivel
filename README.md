Classificador de nível do personagem;
Após derrotar um inimigo "aleatório" o sistema plota um "System Error" e gera o valor que será recebido pela variável xp aleatoriamente;
O valor recebido pela variável deve estar entre 0 e 15000;
Após a variavel receber o valor aleatório ela irá informar a mensagem ao usuário informando a quantidade de XP recebida. Irá repetir o nome do herói e irá informar seu nível;
Se os valores estiverem acima do teto estipulado para o recebimento irá mostrar uma mensagem diferente.
